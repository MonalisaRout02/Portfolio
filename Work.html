Projects

Here are some of my recent projects and a brief information about them.

Restaurant Management System

Developing a relational database management project using SQL, HTML, CSS, PHP, and Javascript. The front end is simple and user-friendly, the website can be used to order food from the menu and also can be used to make table reservations. The database holds the user login details, order details and related information, it is also updated for each transaction.

SwiftFlight

Built a game called "SwiftFlight" which is an addictive and fun 2D game inspired by the popular Flappy Bird. The game features a delightful bird that players control by tapping the spacebar to make it fly through obstacles and collect food particles. It is built using OpenGL along with Freeglut Library.

KickDrum

A fun musical website that is built using HTML, CSS and JavaScript, mainly describing the use of eventListeners in JavaScript.
